# Шрифты TinkoffSans

Поместите следующие файлы шрифтов в эту папку:

1. `TinkoffSans.ttf` - обычное начертание (Regular, 400)
2. `TinkoffSans_Medium.ttf` - среднее начертание (Medium, 500)
3. `TinkoffSans_Bold.ttf` - жирное начертание (Bold, 700)

## Скачивание шрифтов

Шрифты можно скачать по ссылке: https://disk.360.yandex.ru/d/BP8QFFd15fec2A

После скачивания поместите все три файла в эту папку (`frontend/public/fonts/`).

## Использование

Шрифты автоматически подключены через CSS и используются как основной шрифт сайта через Tailwind CSS.
