# FinDER Dataset (Russian Translation)

## Описание

**FinDER** (Financial Document Extraction and Reasoning) — датасет для задачи ответов на вопросы по финансовым документам от Linq-AI-Research.

Датасет содержит вопросы о финансовых отчетах компаний (10-K, годовые отчеты), где для ответа требуется использовать информацию из нескольких референсов (отрывков документов). Каждый вопрос имеет несколько референсов (>= 3), что делает задачу более сложной и требует синтеза информации из разных частей документа.

Переведенный на русский язык вариант оригинального датасета с HuggingFace.

## Структура данных

Каждая запись содержит:

- **text** (str): Вопрос/текст на русском языке
- **references** (list[str]): Список референсов (текстовых отрывков из финансовых документов) на русском языке
- **answer** (str): Ответ на вопрос на русском языке

## Формат

JSON файл с массивом объектов:

```json
[
  {
    "text": "...",
    "references": ["...", "..."],
    "answer": "..."
  }
]
```

## Статистика

- Все записи имеют >= 3 референсов
- Все тексты переведены с английского на русский язык
- Сохранена структура оригинального датасета
